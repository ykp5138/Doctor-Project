# ============================================================
# WINDOWS SETUP — NVIDIA GPU (CUDA 12.1)
# Tested with: Python 3.10, CUDA 12.1, torch 2.5.1+cu121
#
# PREREQUISITES
# -------------
# 1. Install Python 3.10 (not 3.11+, whisperx requires 3.10):
#      https://www.python.org/downloads/release/python-31011/
#    Check: python --version
#
# 2. Install CUDA 12.1 toolkit (if not already installed):
#      https://developer.nvidia.com/cuda-12-1-0-download-archive
#    Check: nvcc --version
#
# 3. Install ffmpeg and add to PATH:
#      https://www.gyan.dev/ffmpeg/builds/  (get ffmpeg-release-full)
#    Unzip, add the bin\ folder to your system PATH
#    Check: ffmpeg -version
#
# 4. Install Ollama and pull model:
#      https://ollama.com/download/windows
#      ollama pull llama3
#
# 5. Create .env in project root:
#      HF_TOKEN=<huggingface token with pyannote access>
#      ASSEMBLYAI_API_KEY=<assemblyai key>
#
# INSTALL STEPS
# -------------
# Open PowerShell in project root:
#
#   python -m venv venv
#   .\venv\Scripts\Activate.ps1
#
#   # Install PyTorch with CUDA 12.1 FIRST (must be before whisperx):
#   pip install torch==2.5.1+cu121 torchaudio==2.5.1+cu121 torchvision==0.20.1+cu121 --index-url https://download.pytorch.org/whl/cu121
#
#   # Then install everything else:
#   pip install -r requirements_windows_cuda.txt
#
# RUN
# ---
#   Terminal 1: ollama serve
#   Terminal 2: uvicorn main:app --reload
#   Terminal 3: cd frontend && npm install && npm run dev
# ============================================================
